## 5. Конкретные правки (раздел 4)

### 5.1 Опыт работы (experience)

- **Первый:** Только текст «Добавить блок experience» и ссылка на рекомендации.
- **Второй:** Полный пример JSON: company, dates, position, description (массив
  достижений). Удобно как шаблон.

### 5.2 Образование

- **Первый:** `education: { main: [...], courses: [...] }` — ГПТУ в main; ATI,
  Zerokoder в courses.
- **Второй:** Плоский массив `education: [...]` с полем `type` (Среднее
  специальное / Повышение квалификации).

### 5.3 Навыки

- **Первый:** JSON `skills_structured`: tools, knowledge, metrics, soft_skills.
- **Второй:** Текстовый список категорий (Программирование и БД, Инструменты,
  Аналитические методы, Продуктовая и маркетинговая аналитика, Процессы).

### 5.4 Summary / О себе

- **Первый:** Поле `summary` в JSON (одна строка).
- **Второй:** Поле `about` в JSON (абзац 3–5 предложений).

### 5.5 Заголовок и salary

- В обоих: исправить salary, убрать слияние с текстом.
- В первом: заголовок «Аналитик данных (Product/Marketing)»; во втором не
  дублируется.
