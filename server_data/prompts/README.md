# Папка промптов

Здесь хранятся **промпты для проведения аудита резюме**.

- **Сейчас:** файлы лежат локально, внутри расширения. Расширение подгружает промпт по умолчанию из `default-audit.txt`; пользователь может править его в настройках (значение сохраняется в браузере).
- **В дальнейшем:** планируется загрузка промпта из **удалённого места** (URL), чтобы обновлять текст аудита без переустановки расширения.

## Файлы

- `default-audit.txt` — промпт по умолчанию. В тексте используется плейсхолдер `RESUME_JSON`: при отправке в ИИ он заменяется на JSON резюме.
- `programmer-code-review.txt` — промпт для проверки проекта как программист: поиск ошибок синтаксиса, логики, обратная связь по коду и архитектуре.
