# Как обновить расширение после изменения кода

## Шаг 1: Перезагрузить расширение в Chrome

1. Откройте **chrome://extensions/** в адресной строке
2. Найдите расширение **"Аудит резюме (RASHOTKLIK)"**
3. Нажмите кнопку **"Обновить"** (значок ⟳ или круговая стрелка)

## Шаг 2: Обновить страницу с резюме

1. Вернитесь на вкладку с открытым резюме на **hh.ru**
2. Нажмите **F5** или Ctrl+R (обновить страницу)
   - Это важно! Без обновления страницы будет работать старая версия content script

## Шаг 3: Попробовать получить резюме

1. Откройте **popup расширения** (клик по иконке)
2. Нажмите кнопку **"Получить резюме с HH.ru"**

## Отладка

### Если не работает, откройте консоль браузера:

1. На странице резюме нажмите **F12** (Developer Tools)
2. Перейдите на вкладку **Console**
3. Обновите страницу (**F5**)
4. Снова нажмите "Получить резюме" в popup
5. В консоли должны появиться сообщения вида:
   ```
   [RESUME_PARSER] === Starting resume extraction ===
   [RESUME_PARSER] URL: https://...
   [RESUME_PARSER] Collecting name...
   [RESUME_PARSER] Found h1: ...
   ...
   [RESUME_PARSER] === Extraction successful ===
   ```

### Если видите ошибки или неполные данные:

- Сделайте **скриншот консоли** и покажите его
- Проверьте, что URL содержит `/resume/`
- Убедитесь, что страница полностью загружена (не показывает индикатор загрузки)

## Выключить режим отладки

После того как всё заработает, можно выключить DEBUG логи:

1. Откройте файл `scripts/contentScript.js`
2. Найдите строку: `const DEBUG = true;`
3. Измените на: `const DEBUG = false;`
4. Повторите **Шаг 1 и 2** выше
